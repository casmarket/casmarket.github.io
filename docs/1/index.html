夏キャスマーケットとは?

2020年、現実での制約を受け付けないバーチャル空間の広がりは著しく、 VRChatでのバーチャルマーケットも、規模は倍々に大きくなっています。 バーチャルキャストでは近い将来、アバター、アイテムを完全バーチャル空間内でやり取りする 仕組みと通貨が実装される見込みです。

さて、そんなバーチャルキャストでは未だ開催されてなかったマーケットですが、 夏キャスマーケット2020を開催します！ 新型コロナによって自粛が続く世の中ですが、この夏はお祭り気分で バーチャル空間を楽しみましょう。 バーチャルキャストのクリエイターと、バーチャルで生きる皆さんの架け橋となりますように……。

※バーチャルキャスト非公式イベントです。

データ規定

3D

  • 3DアイテムはVCI形式で5MiB以下になること。
  • 大きさは15m×15m×高さ50mを上限とする。(ただし大きいほど背景端に配置されやすい)
  • 1つのVCI中にSubItemの数は10個までで、必要以上にコンポーネントを使用しない。
  • マテリアル数の上限は10個までで、不透明 (RenderType「Opaque」) を推奨。
  • SubItem名は極力被りにくい名前にする。(合わせた際に不具合が発生するため)
  • スクリプト、オーディオ、エフェクトは使用しない。
  • 出展者情報は別途キャプションにアイコン・名前を記載。

アイテム

  • 手持ちアイテムを想定。
  • 80cmの土台が付く。

アバター

  • VCI形式でポーズを付けた状態で提出。

ジオラマ

  • 出店〜建物規模のサイズはすべてこちら。

2D

  • 2Dファイルは画像形式で1MiB以下になること。(2枚の場合は合計で1MiB以下)
  • 大きさは 1024×1024px 以内推奨で、アスペクト比は自由。
  • 設置場所によって設置サイズは適宜変更される。
  • 半透明は使用不可。透明は可。(マテリアルでCutout使用予定のため)
  • 出展者情報は別途キャプションを用意しないため、すべてポスターに書き込むこと。

ポスター

  • 1枚で提出。

立体ポスター

  • 2枚で提出。
  • 重ねて立体風に。
