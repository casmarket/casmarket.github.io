秋キャスマーケットとは?

世間ではソーシャルディスタンスを意識した新たな日常を受容する時流となり、 非接触でコミュニケーションがとれるVRへより一層関心が高まっています。 バーチャルキャストでは、バーチャル空間内でアバター、アイテムを個人で売買できる 「ユーザストア」の公開が目前となっています。

「夏キャスマーケット2020」の開催から早3ヶ月となりますが、 バーチャルキャスト2回目のユーザ開催マーケット『秋キャスマーケット2020』を開催します！ バーチャルキャストのクリエイターと、バーチャルで生きる皆さんの架け橋となりますように……。

※バーチャルキャスト非公式イベントです。

データ規定

VCI

  • 3Dアイテムは VCI形式 で「THE SEED ONLINE」に商品として公開すること。
  • 提出から 11/9 まで公開状態を保つこと。
